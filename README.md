Coding Challenge for JAM Front-end Developer



<h3>To start app:</h3>

Run `npm start` in the terminal and open [http://localhost:3000](http://localhost:3000) to view it in the browser.

In another tab, run `node server.js` to start back-end server built to circumvent CORS issue with API.
